Responsive Design: Alkalmazkodó weboldalak

Mi az a reszponzív design?

A reszponzív design azt jelenti, hogy weboldalunk automatikusan alkalmazkodik a különböző képernyőméretekhez (mobil, tablet, desktop). Ez ma már alapvető elvárás minden weboldallal szemben.

Viewport meta tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Query-k használata

/* Mobilon */
@media (max-width: 768px) {
    .container {
        width: 100%;
        padding: 10px;
    }
}

/* Tableten */
@media (min-width: 769px) and (max-width: 1024px) {
    .container {
        width: 90%;
        padding: 20px;
    }
}

/* Asztali gépen */
@media (min-width: 1025px) {
    .container {
        width: 80%;
        max-width: 1200px;
        padding: 30px;
    }
}

Rugalmas mértékegységek

  • em - A szülő elem betűméretéhez viszonyított méret
  • rem - A root (html) elem betűméretéhez viszonyított méret
  • % - A szülő elem méretéhez viszonyított százalékos méret
  • vw/vh - A viewport szélességéhez/magasságához viszonyított méret

Reszponzív képek

img {
    max-width: 100%;
    height: auto;
}

/* Különböző képek különböző méretekhez */
<picture>
    <source media="(min-width: 1024px)" srcset="nagy.jpg">
    <source media="(min-width: 768px)" srcset="kozepes.jpg">
    <img src="kicsi.jpg" alt="Reszponzív kép">
</picture>

Gyakorlati feladat
