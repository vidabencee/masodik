Box Model: Az elemek méretezésének alapjai

Mi az a Box Model?

A Box Model a webfejlesztés egyik alapvető koncepciója. Minden HTML elem egy "doboznak" tekinthető, amely különböző rétegekből áll: tartalom (content), belső margó (padding), keret (border) és külső margó (margin).

A Box Model részei

/* Box Model példa */
.doboz {
    /* Tartalom mérete */
    width: 200px;
    height: 100px;
    
    /* Belső margó */
    padding: 20px;
    
    /* Keret */
    border: 2px solid black;
    
    /* Külső margó */
    margin: 10px;
}

Box Model tulajdonságok részletesen

  • padding - Belső térköz az elem tartalma és kerete között
  • border - Az elem kerete
  • margin - Külső térköz az elem és a szomszédos elemek között
  • width/height - Az elem szélességi és magassági mérete

Box-sizing tulajdonság

/* Az elem teljes mérete tartalmazza a padding-ot és border-t */
.doboz {
    box-sizing: border-box;
}

/* Alapértelmezett: csak a tartalom méretét veszi figyelembe */
.doboz {
    box-sizing: content-box;
}

Gyakorlati feladat

Készíts három egymás melletti doboz elemet! Mindegyiknek állíts be különböző padding, margin és border értékeket. Figyelj arra, hogy egyenlő távolságra legyenek egymástól!
