Mi az a HTML?

A HTML (HyperText Markup Language) a weboldalak építőköve. Ez egy jelölőnyelv, amely meghatározza a weboldal struktúráját és tartalmát. Gondolj rá úgy, mint egy ház vázára - ez adja meg az alapvető szerkezetet.

Alapvető HTML elemek

Nézzük meg a legalapvetőbb HTML elemeket:

<!DOCTYPE html>
<html>
<head>
    <title>Az oldal címe</title>
</head>
<body>
    <h1>Ez egy főcím</h1>
    <p>Ez egy bekezdés.</p>
</body>
</html>

A legfontosabb HTML tagek

  • <h1> - <h6> - Címsorok
  • <p> - Bekezdés
  • <a> - Hivatkozás
  • <img> - Kép

Gyakorlati feladat

Készíts egy egyszerű HTML oldalt, amely tartalmazza a nevedet címsorként, és írj magadról egy rövid bemutatkozást bekezdésként!
