Pozicionálás: Elemek elhelyezése a weboldalon

Position tulajdonság

A CSS position tulajdonság segítségével határozhatjuk meg, hogyan helyezkedjen el egy elem a weboldalon. Öt különböző értéket vehet fel: static, relative, absolute, fixed és sticky.

Pozicionálási módok

/* Alapértelmezett pozicionálás */
.static {
    position: static;
}

/* Relatív pozicionálás */
.relative {
    position: relative;
    top: 20px;
    left: 20px;
}

/* Abszolút pozicionálás */
.absolute {
    position: absolute;
    top: 0;
    right: 0;
}

/* Fix pozicionálás */
.fixed {
    position: fixed;
    bottom: 20px;
    right: 20px;
}

/* Sticky pozicionálás */
.sticky {
    position: sticky;
    top: 0;
}

Mikor melyiket használjuk?

  • Static: Alapértelmezett, a normál dokumentumfolyamban marad
  • Relative: Az eredeti helyéhez képest mozdíthatjuk el
  • Absolute: A legközelebbi pozicionált szülőhöz képest helyezkedik el
  • Fixed: A viewport-hoz képest fix pozícióban marad
  • Sticky: Görgetésnél "ragadós" viselkedést mutat

Z-index használata

A z-index tulajdonság segítségével szabályozhatjuk az elemek egymáshoz viszonyított rétegrendjét:

.felso-reteg {
    position: relative;
    z-index: 2;
}

.also-reteg {
    position: relative;
    z-index: 1;
}

Gyakorlati feladat

Készíts egy oldalt, ahol van egy fix fejléc, amely mindig látható marad görgetéskor, és egy "vissza a tetejére" gomb, ami fixed pozícióban van a jobb alsó sarokban!
