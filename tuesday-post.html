CSS Bevezetés: Stílusok világa

Mi az a CSS?

A CSS (Cascading Style Sheets) egy stílusleíró nyelv, amely a HTML elemek megjelenését határozza meg. Ha a HTML a ház váza, akkor a CSS a festés, a bútorok és a dekoráció.

CSS használata

A CSS-t háromféleképpen használhatjuk:

/* 1. Inline stílus */
<p style="color: red;">Piros szöveg</p>

/* 2. Internal CSS a head részben */
<style>
    p {
        color: red;
    }
</style>

/* 3. External CSS külön fájlban */
/* styles.css */
p {
    color: red;
}

Alapvető CSS tulajdonságok

  • color - Szöveg színe
  • background-color - Háttérszín
  • font-size - Betűméret
  • font-family - Betűtípus

CSS szelektorok

/* Elem szelektor */
p {
    color: blue;
}

/* ID szelektor */
#egyedi {
    font-size: 20px;
}

/* Class szelektor */
.kiemelt {
    background-color: yellow;
}

Gyakorlati feladat

Készíts egy HTML oldalt, amelyen van egy címsor és három bekezdés. A címsort tedd pirossá, az első bekezdést kékké, a másodikat zölddé, és a harmadikat sárgává!
